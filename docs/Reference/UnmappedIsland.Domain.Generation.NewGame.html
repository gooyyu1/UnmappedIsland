    Class NewGame

    新しいゲームの開始一式（world/プレイヤーの生成 → 地形生成 → 島の実体化 → プレイヤー配置）を 1回の呼び出しに閉じ込める入口。呼び出し側（UnityのGameManager等）は「Codexとシードを渡す」 だけでよく、生成と配置の手順・順序を知らなくてよい（自分のことは自分でする、CLAUDE.md参照）。
