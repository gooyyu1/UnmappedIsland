    Class Path

    道（locations.yamlのpath object_def）に対する、UI/ゲームロジック向けの型付きビュー。 World/PlayerCharacter/Location と同じ理由で継承ではなくラップにしている。

    travel_minutes/required_progress/destination_idはいずれも生成時（IslandSpawner）にインスタンス ごとへ上書きされる値であり、このビューは読み取りとtravelアクションの実行だけを提供する。 移動そのもの（actorの所属差し替え・時間消費）はYAML側のtravelアクション（move+duration）が担う。
