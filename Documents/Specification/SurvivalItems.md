# 道具・武器・容器・衣類 アイテム案

## 概要

一人でのサバイバルという前提のもと、実在の原始技術を参考にしつつ、過度に高度な技術（紡績・機織り、製鉄など）は
除外し、「手が出せる範囲で便利・快適なもの」に絞ったアイテム案です。グレードは各カテゴリ2〜4段階に留め、
必要な素材の種類が増えすぎないよう、素材を複数のアイテムで使い回しています。

本ドキュメントは検討結果であり、確定仕様書ではありません。`RecipeSystem.md` で検討したレシピの仕組み
（工程・素材要求）を前提としていますが、本書では具体的な工程分割やYAMLの記法までは踏み込まず、
「何を、何から作るか」という内容面の整理に留めています。

## 0. 素材一覧

このアイテム群全体で使う原材料は、以下の9種類（＋既存の食料資源2種類の流用）に絞っています。

| 素材 | 主な入手方法 | 用途 |
|---|---|---|
| 石 | 採取 | 刃物・打撃部（無加工〜柄付き） |
| 木 | 採取・伐採 | 柄・軸・掘り棒・槍の柄 |
| 植物繊維 | 採取 | 紐（柄の固定・縫製）、編み材 |
| 生皮 | 狩猟 | 衣類・袋（未処理、劣化しやすい） |
| 樹皮 | 採取 | なめし剤（生皮→なめし革への加工に使用） |
| 獣骨・角 | 狩猟 | 掘削具の頭部（シャベル・ピック） |
| 銅鉱石 | 採掘 | 青銅の材料 |
| 錫鉱石 | 採掘 | 青銅の材料 |
| 粘土 | 採取 | 土器 |
| （ヤシの実） | 既存の食料資源を流用 | 容器（無加工） |
| （竹） | 既存の資源を流用と想定 | 容器（無加工） |

**なめし革**（生皮＋樹皮）と**青銅**（銅鉱石＋錫鉱石）は、それ自体を他のレシピの材料として使う**中間素材**として扱います。

## 1. 道具（採取・加工用）

| グレード | アイテム | 素材（レシピ） | 備考 |
|---|---|---|---|
| 1 | ただの石 | 石 | 加工なし。叩き石として使用 |
| 1 | 大きな石 | 石 | 数回打ち欠いただけの、柄なしで握って使う石器 |
| 1 | 石の刃物（ナイフ） | 石 | 剥片をそのまま使用。柄は不要 |
| 2 | 石の斧 | 石 + 木 + 植物繊維 | 柄付け技術（紐で固定）が必要 |
| 2 | 石のハンマー | 石 + 木 + 植物繊維 | 石の斧と同じ柄付け技術。打撃部の形状違い |
| 3 | 青銅の斧 | 青銅 + 木 + 植物繊維 | 石の斧と同じ柄付け技術で頭部を青銅に置き換え |
| 3 | 青銅のナイフ | 青銅 + 木 + 植物繊維 | 柄をつけて強化した上位のナイフ |

## 2. 掘削具

| グレード | アイテム | 素材（レシピ） | 備考 |
|---|---|---|---|
| 1 | 火で硬化させた掘り棒 | 木 | 先端を尖らせ火で炙って硬化。木の槍と同じ技術 |

## 3. 武器

| グレード | アイテム | 素材（レシピ） | 備考 |
|---|---|---|---|
| 1 | 木の槍 | 木 | 先端を尖らせ火で炙って硬化。掘り棒と同じ技術 |
| 2 | 石鏃の槍 | 木 + 石 + 植物繊維 | 石の刃物を柄付け技術で穂先として固定 |
| 3 | 青銅の穂先の槍 | 木 + 青銅 + 植物繊維 | 同じ柄付け技術で穂先を青銅に |

石のハンマーは打撃武器（棍棒相当）としても兼用できるため、武器専用の棍棒は別途用意していません。

## 4. 容器

| グレード | アイテム | 素材（レシピ） | 備考 |
|---|---|---|---|
| 1 | ヤシの実の器 | ヤシの実 | 半分に割るだけ。多少の防水性あり |
| 1 | 竹筒 | 竹 | 節の間を切り出すだけ。水を運ぶのに優秀 |
| 2 | 皮袋 | 生皮 + 植物繊維（縫製） | 生皮のままなら劣化・腐敗しやすい |
| 2 | 皮袋（なめし革） | なめし革 + 植物繊維（縫製） | 同じ形状だが長持ち。なめし革を使った上位互換 |
| 3 | 土器（本焼き） | 粘土 | 高温での焼成が必要。真に長持ちし水も保持できる |

**天日干しの素焼き粘土は水には向きません**（濡れると崩れる）。土器を作るなら本焼きまで行う前提とし、
素焼きの中間アイテムは別途用意していません。かご（編み物）は固形物の運搬・保管に使えますが、防水加工は
本書のスコープでは扱わず（7節参照）、水を保持する容器としては数えていません。

容器自体が耐火性を持たなくても、焼け石を投入する「石焼き調理」で煮炊きは可能という想定です（詳細な調理系の
仕組みは別途検討）。

| グレード | アイテム | 素材（レシピ） | 備考 |
|---|---|---|---|
| 1 | 束ねた葉の衣類 | 植物繊維 | 加工ほぼゼロ |
| 1 | 生皮の衣類 | 生皮 | 縫製なしでそのまま身につける想定。劣化しやすい |
| 2 | 編んだ葉の衣類 | 植物繊維 | 編み技術が必要 |

## 6. 素材の再利用状況

各素材が何種類のアイテムに使われているかの一覧です。

| 素材 | 使用アイテム数（概算） |
|---|---|
| 石 | 4（無柄3種＋石の斧の材料） |
| 木 | 6（柄付き道具・武器・掘り棒・槍） |
| 植物繊維 | 8（柄の固定・縫製・編み物・衣類の大半） |
| 生皮 | 2（皮袋・衣類。なめし革の元にもなる） |
| 樹皮 | 1（なめし革への加工にのみ使用） |
| 獣骨・角 | 2（シャベル・ピック） |
| 銅鉱石／錫鉱石 | 4（青銅の斧・ナイフ・シャベル・槍の材料） |
| 粘土 | 1（土器） |
| ヤシの実／竹 | 各1（既存資源の流用） |

木・植物繊維・石が多くのアイテムで共有される中心素材となり、専用素材（樹皮・粘土）は最小限に抑えられています。

## 7. 除外・保留にした要素

以下は、今回のトリミング方針（一人でのサバイバル・素材種類の抑制）に基づき、本書のスコープから意図的に外した
要素です。将来的に拡張したくなった場合の候補として記録しておきます。

- **布（紡績＋機織り）**: 紡ぐ・織るという専用の技術・道具（紡錘、織機）一式が必要で、なめし革までで十分快適性を
  確保できるため、今回は対象外としました。
- **鉄**: 精錬に必要な高温・炉の技術が、一人のサバイバルでは非現実的と判断し、青銅を上限としました。
- **柄付け・容器の防水における樹脂（松脂等）の使用**: 実在の技法では紐に加えて樹脂も使われますが、紐（植物繊維）
  だけでも柄付けは成立するため、素材の種類を減らす目的で樹脂は使わない前提にしました。これに伴い、防水加工した
  かご（編みかごへの樹脂塗布）も本書のアイテム一覧からは外しています。
- **ひょうたん（gourd）容器**: ゲーム内の植生に含めるかどうかが未確定なため、今回は保留としました。
- **自然銅の冷間鍛造品**: 精錬前の自然銅を叩いて加工する、銅よりさらに手前の段階も実在しますが、グレード数を
  抑えるため、銅そのものを個別アイテム化せず、青銅の材料（銅鉱石）としてのみ扱っています。

## 8. 未決事項

- 各アイテムの具体的な効果値（防御力・耐久値・重さ・サイズなど）は未定義。`GameElementDefinition.md` の
  `props`/`passive`（6節・8節）・`ContainerSystem.md` のサンプルに準じて別途定義する
- `RecipeSystem.md` の `steps`（工程分割）・`duration`（所要時間）を、本書のどのアイテムにどう割り当てるか
- なめし革・青銅という中間素材が、`RecipeSystem.md` の「成果物の `object_defs` にレシピを埋め込む」設計上、
  それ自体を1つの完成品（`object_defs`）として扱ってよいか（中間素材も「レシピで作られるアイテムの一種」として
  扱えば、既存の設計をそのまま流用できるはずだが未確認）
