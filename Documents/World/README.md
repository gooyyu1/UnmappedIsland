# ワールド（実際にゲームに登場する内容）

`Engine/` の文法・汎用サブシステムを使って、**このゲームに実際に登場する具体的な内容**をどう表現するかを
まとめたドキュメントを格納します。気候、地形の生成、探索、道具・武器・容器といったアイテムなど、
プレイヤーが実際に目にする世界の中身がここに含まれます。

文法そのもの（`traits`/`props`/`slots`/`actions` 等）や、どんな内容にも適用できる汎用サブシステム
（レシピ・コンテナ・耐久値）は含みません。そちらは [`../Engine/`](../Engine/README.md) を参照してください。

## 収録ドキュメント

- [気候システム設計](./ClimateSystem.md)
- [地形生成システム設計](./TerrainGeneration.md) — 島の座標・軸・LocationTypeマッチング・パスネットワーク生成のアルゴリズム
- [探索・道システム設計](./ExplorationSystem.md) — 生成された土地のスロット構成・探索・道の発見と移動
- [道具・武器・容器・衣類 アイテム案](./SurvivalItems.md)

## 含まれるドキュメント例

- 気候・季節・天候
- 地形生成・マップ構成
- 探索・移動などのゲームプレイ挙動
- アイテム・道具・武器・容器・衣類などの内容案
- 敵・NPC・生態系（今後追加予定）
